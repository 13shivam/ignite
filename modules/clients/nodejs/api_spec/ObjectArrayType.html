Class: ObjectArrayType

ObjectArrayType(elementType)

Class representing an array of Ignite complex type objects.

It is described by COMPOSITE_TYPE.OBJECT_ARRAY ObjectType.COMPOSITE_TYPE and by a concrete ComplexObjectType instance which defines an element of the array.
